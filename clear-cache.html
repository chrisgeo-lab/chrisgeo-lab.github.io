🗑️ Clear RouteFlow Cache

If you're seeing old bugs after updates, the browser is serving cached files. Click below to force-clear everything:

Manual Steps

  1. Chrome/Edge: Press Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
  2. Select "All time" and check "Cached images and files"
  3. Click "Clear data"
  4. Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)

Developer Console

  1. Open DevTools (F12)
  2. Go to Application tab
  3. Click Service Workers → Unregister
  4. Click Cache Storage → Delete all caches
  5. Reload page
